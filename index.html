MISU pyFotoResize

Hromadná úprava velikosti fotografií s možností přidání vodoznaku

Download this project as a .zip file Download this project as a tar.gz file

MISU pyFotoResize je Open-source aplikace vyvinutá v jazyku Python 3, jejíž uživatelské rozhraní (GUI) bylo postaveno na komponentách velmi známého bindingu pro Qt a tím je PySide. Díky těmto vlastnostem se řadí nejen do kategorie multiplatformní, tedy spustitelné na různých operačních systémech (testováno na platformách Linux a Windows), ale také do kategorie ZDARMA, volně ke stažení.
